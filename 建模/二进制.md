# 二进制


比如今天遇到的一个问题，字符串 str 由 '0' 和 '1' 组成，但是原始的状态可能是 '1111' 现在要改成 '1010' 或者'0101' 问修改成这样的结果的最少操作数

我最开始想着就是，第一个位置的是固定的，然后按照这个结构变化，但是它的形态由两种结果，可能另一种结果所需的操作数更少，但是呢这两种操作数的总数 是n  所以返回值 应该是 return min(s.size() - ans, ans)

```cpp

class Solution {
public:
    int minOperations(string s) {
        int ans = 0;
        for(int i = 1; i < s.size(); i++){
            if(s[i] - s[i - 1]  == 0){
                if(s[i] == '0'){
                    s[i] = '1';
                }else if(s[i] == '1'){
                    s[i] = '0';
                }
                ans++;
            }
        }
        return ans;      //  return min(s.size() - ans, ans)  这样才正确
    }
};


```

```cpp
class Solution {
public:
    int minOperations(string s) {
        /*
            s 的结果只有两种，而且 改成cnt 1需要的操作数为x，那么改成cnt2需要的操作数为 (n - x)
        */
        int n = s.size();

        int cnt = 0;
        for(int i = 0; i < n; i++){
            char expected = (i % 2 == 0) ? '0' : '1';
            if(s[i] == expected){
                cnt++;
            }
        }
        return min(n - cnt, cnt);
    }
};

```
