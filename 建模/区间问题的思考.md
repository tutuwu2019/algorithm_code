# 区间问题的思考

> 事实上区间问题，本质是排列组合，但是已经有成熟的算法俗称惯性思维，能够很快过滤数据，但是我有点轴，想把每个细节都记录下来。


引子：leetcode 57 插入区间

```text
给你一个 无重叠的 ，按照区间起始端点排序的区间列表 intervals，其中 intervals[i] = [starti, endi] 表示第 i 个区间的开始和结束，并且 intervals 按照 starti 升序排列。同样给定一个区间 newInterval = [start, end] 表示另一个区间的开始和结束。

在 intervals 中插入区间 newInterval，使得 intervals 依然按照 starti 升序排列，且区间之间不重叠（如果有必要的话，可以合并区间）。

返回插入之后的 intervals。

注意 你不需要原地修改 intervals。你可以创建一个新数组然后返回它。

 

示例 1：

输入：intervals = [[1,3],[6,9]], newInterval = [2,5]
输出：[[1,5],[6,9]]
示例 2：

输入：intervals = [[1,2],[3,5],[6,7],[8,10],[12,16]], newInterval = [4,8]
输出：[[1,2],[3,10],[12,16]]
解释：这是因为新的区间 [4,8] 与 [3,5],[6,7],[8,10] 重叠。
 

提示：

0 <= intervals.length <= 104
intervals[i].length == 2
0 <= starti <= endi <= 105
intervals 根据 starti 按 升序 排列
newInterval.length == 2
0 <= start <= end <= 105
```

建模：
整个处理过程分为三个清晰的物理过程：
1. 左侧无重叠区：这些区间在待插入区间的左边，完全没有交集，直接加入结果
2. 中间叠合病区：不断更新 newInterval 的边界，newStart = min(newStart, interval.start)  newEnd = max(newEnd, interval.end())
3. 右侧无重叠区：先将合并后的 newInterval 加入结果，再把剩余的区间全部加入。(注意到这里，已经把 newInterval 以插入和维护了，后面要做的，就是把未污染的 右侧加入结果中)


> 一个随着时间推进的”流水线“
>> 建模核心：分治筛选

1. 原有数据排序，形成"时空轴"
2. 三段式拆解
  - 左侧完全不重叠
  - 合并重叠区间
  - 右侧完全不重叠区间


这样，无论插入的是哪种情况都会被塞进去。
比如，新插入的在是在 最左边，那么就是 第三种情况。
比如，新插入的实在 最右边，那么就是第一种情况。
第二种情况是稍微复杂的，因为在中间插入，可能会造成二次合并(也可以理解为二次插入) 但是这种合并也不可能是是第一种情况(当前的 ans 已经是暂定的，不会被混乱)(可能是第三种情况，比如合并后 intervals 后面没有元素了)

第一种情况如何建模？ 就是 newInterval 必然是在 intervals[i] 的右边，那么 intervals[i][1] < newInterval[0]

所以这样操作
```cpp
while(i < n && intervals[i][1] < newInteral[0]){
  ans.push_back(intervals[i++]);
}
```

接着处理的就是，合并重叠、右侧完全不重叠

在 intervals[i][1]>= newInterval[0] 的情况下讨论分析了(这已经表明有交集、或者完全飞出去了)

> 完全飞出去了是什么情况呢？ intervals[i][0] > newInterval[1]

>> 那么再取反，这不就是第二种情况了吗？
```cpp

// 因为是层层过滤，所以就把隐含的条件给隐藏了，如果要合并重叠区间，完整的过滤条件是
// newInterval[0] <= intervals[i][1] && newInterval[1] >= intervals[i][0]

while(i < n && intervals[i][0] <= newInterval[1])    // 同上分析，  while(i < n &&  newInterval[0] <= intervals[i][1] && newInterval[1] >= intervals[i][0]) 也是可以的
  newInterval[0] = min(intervals[i][0], newInterval[0]);
  newInterval[1] = max(intervals[i][1], newInterval[1]);
  i++;
}
ans.push_back(newInterval);

```

我再举一个例子，固化
intervals = {[10, 12]}

newInterval = [11, 14]

newInterval[0] <= intervals[i][1] && newInterval[1] >= intervals[i][0]
11           <=       12        &&      14 >= 10

[10 11 12 14] ==> [min(10, 11), max(12, 14)] ==> [10, 14] 

到这里，右侧完全不重叠区间，就好处理了。
> 隐含意思，newInterval 已经全部都处理好了，结构也维护好了，剩下的就是把剩下的元素插到屁股后面

```cpp
while(i < n){
  ans.push_back(intervals[i++);
}

```
